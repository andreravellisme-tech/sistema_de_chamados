⚙️ Configure o Firebase

Abra o arquivo HTML num editor de texto e substitua os campos COLE_AQUI_... pelos dados do seu projeto.

  1. Acesse console.firebase.google.com → crie um projeto
  2. Ative o Firestore Database → modo teste → região southamerica-east1
  3. Ative o Authentication → Sign-in method → Email/Senha → Ativar
  4. Adicione app Web (</>) → copie o firebaseConfig → cole no HTML
  5. Aplique as Regras do Firestore descritas no arquivo LEIA-ME.md
  6. Crie o primeiro usuário Admin via Firebase Auth e execute setup-admin.js

📄 Leia o arquivo LEIA-ME.md para instruções completas de segurança.
